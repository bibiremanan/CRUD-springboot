# SpringBoot Task

### Technologies: 

- Java8
- Spring boot
- Spring JPA 
- Database(H2/My Sql)

 

### Create an entity class  Product with following properties
 
```sql
   Id - [long] - primary key

   Name - [String]

   Description - [String]

   Price - [double]
```

- For getting all products from db
